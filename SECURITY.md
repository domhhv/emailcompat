# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing **domhryshaiev@gmail.com**.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Resolution**: Depends on severity and complexity

Please do not open public issues for security vulnerabilities.

## Scope

This policy applies to the Email Client Previewer repository and its deployed instances.
