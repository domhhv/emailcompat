# Contributing

Thanks for your interest in contributing to Email Client Previewer!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/domhhv/email-previewer.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

```bash
npm run dev             # Start dev server
npm run build           # Build for production
npm run typecheck       # Run TypeScript checks
npm run eslint:check    # Run ESLint
npm run prettier:check  # Check formatting
```

## Commit Messages

We use [Conventional Commits](https://www.conventionalcommits.org/). Examples:

- `feat: add dark mode support`
- `fix: resolve preview rendering issue`
- `docs: update readme`

## Pull Requests

1. Update documentation if needed
2. Ensure all checks pass
3. Keep PRs focused on a single change
4. Link any related issues

## Questions?

Open an issue if you have questions or need help getting started.
